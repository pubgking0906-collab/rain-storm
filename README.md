# XRain

**Turn Predictions Into Profits**

XRain is a blockchain-native prediction market platform powered by Rain Protocol on Arbitrum.

## Features

- 🎯 Trade on real-world outcomes
- ⚡ Instant settlements via smart contracts
- 💰 Earn from accurate predictions
- 🔐 Self-custody wallet integration
- 🌐 Decentralized and transparent

## Live Site

- Production: https://xrain.ai
- Test: https://test.xrain.ai
- Dev: https://dev.xrain.ai

## Tech Stack

- Next.js 16
- Rain Protocol SDK
- Wagmi + Reown (WalletConnect)
- Tailwind CSS
- TypeScript

## Development

\`\`\`bash
npm install
npm run dev
\`\`\`
